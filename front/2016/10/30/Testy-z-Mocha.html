Test z Mocha

October 30th, 2016


Framework do Testowania

Mocha jest frameworkiem do testowania w przeglądarce oraz pod NodeJS. Tak wyglądają przykładowe testy z oficjalnej dokumentacji. Ten zestaw akurat odpala się w przęglądarce, ale ja napiszę test odpalany w NodeJS.

Będę potrzebować nowego projektu z zainstalowanym Mocha.

$ mkdir przykladowe-testy-w-Mocha
$ cd przykladowe-testy-w-Mocha
$ npm init -y
Wrote to przykladowe-testy-w-Mocha/przykladowe-testy-w-Mocha/package.json:
{
  "name": "przykladowe-testy-w-Mocha",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}
$ npm install mocha --save


Test

Wykonanie zestawu testów odpalamy w katalogu z projektem za pomocą polecenia mocha co powinno w tej chwili zakończy się błędem, bo przeież nie napisałem jeszcze żadnego scenariusza. 

$ mocha
Warning: Could not find any test files matching pattern: test



Tworzę katalog tests i za pomocą edytora umieszczam tak plik o nazwie pierwszyTest.js z następującą treścią: 

var assert = require('assert');
describe('Array', function() {
  describe('#indexOf()', function() {
    it('should return -1 when the value is not present', function() {
      assert.equal(-1, [1,2,3].indexOf(4));
    });
  });
});




$ mocha

  Array
    #indexOf()
      ✓ should return -1 when the value is not present


  1 passing (8ms)


Gotowe

Dla wygody w pracy codziennej, podepenijmy jeszcze to polecenie pod NPM. Tak aby w przyszłości można było łatwo odpalić test w danym projekcie za pomoca npm test.

W tej chwili odpalenie testów za pośrednictwem npm będzie kończyć sie błędem: 

$ npm test
$ 
$ > przykladowe-testy-w-Mocha@1.0.0 test przykladowe-testy-w-Mocha/przykladowe-testy-w-Mocha
$ > echo "Error: no test specified" && exit 1

Error: no test specified
npm ERR! Test failed.  See above for more details.



Zmieńmy to, modyfikując plik package.json na wzór: 

  "scripts": {
    "test": "mocha"
  },



Po zapisaniu zmian, NPM powienień wiedzieć że testy powinien wywoływać za pomocą mocha: 

npm test

> przykladowe-testy-w-Mocha@1.0.0 test /home/piotrmroczek/Blog/przykladowe-testy-w-Mocha/przykladowe-testy-w-Mocha
> mocha



  Array
    #indexOf()
      ✓ should return -1 when the value is not present


  1 passing (7ms)
